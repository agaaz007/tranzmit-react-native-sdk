# @tranzmit/shared

Shared internals for the Tranzmit SDKs: HTTP client, identity resolution, config/spec types, integrity validation, and event batching.

This package is an **internal dependency** of [`@tranzmit/react-native`](https://www.npmjs.com/package/@tranzmit/react-native) and is installed automatically. You normally do not depend on it directly.

Full documentation: https://github.com/agaaz007/tranzmit-react-native-sdk#readme
